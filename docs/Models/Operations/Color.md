# Color

The color of the tag. If not provided, a random color will be used from the list: red, yellow, green, blue, purple, pink, brown.


## Values

| Name     | Value    |
| -------- | -------- |
| `Red`    | red      |
| `Yellow` | yellow   |
| `Green`  | green    |
| `Blue`   | blue     |
| `Purple` | purple   |
| `Pink`   | pink     |
| `Brown`  | brown    |
