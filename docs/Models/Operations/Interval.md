# Interval

The interval to retrieve analytics for. Takes precedence over start and end. If undefined, defaults to 24h.


## Values

| Name            | Value           |
| --------------- | --------------- |
| `TwentyFourh`   | 24h             |
| `Sevend`        | 7d              |
| `Thirtyd`       | 30d             |
| `Ninetyd`       | 90d             |
| `Ytd`           | ytd             |
| `Oney`          | 1y              |
| `All`           | all             |
| `AllUnfiltered` | all_unfiltered  |
