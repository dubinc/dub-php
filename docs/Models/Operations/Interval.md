# Interval

The interval to retrieve analytics for. If undefined, defaults to 24h.


## Values

| Name          | Value         |
| ------------- | ------------- |
| `TwentyFourh` | 24h           |
| `Sevend`      | 7d            |
| `Thirtyd`     | 30d           |
| `Ninetyd`     | 90d           |
| `Oney`        | 1y            |
| `Mtd`         | mtd           |
| `Qtd`         | qtd           |
| `Ytd`         | ytd           |
| `All`         | all           |
