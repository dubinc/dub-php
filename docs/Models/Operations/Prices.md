# Prices

Price details for the domain. Will be null if the domain is not available.


## Fields

| Field                                         | Type                                          | Required                                      | Description                                   |
| --------------------------------------------- | --------------------------------------------- | --------------------------------------------- | --------------------------------------------- |
| `registration`                                | *float*                                       | :heavy_check_mark:                            | The domain's registration price in USD cents. |
| `renewal`                                     | *float*                                       | :heavy_check_mark:                            | The domain's renewal price in USD cents.      |
